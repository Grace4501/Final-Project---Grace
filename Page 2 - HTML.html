Example

This is brought up through a short cut: shift + ! and pressing enter

  • !DOCTYPE html: specifies that the document is HTML
  • html lang="en": specifies that the decument is using the english language
  • head: there are two main parts of a website, head is all of the stuff before the elements of the document are put in. It mostly acts as a settings of the document.
  • title: the title, is you guessed it, where you put the title of the document
  • body: The second main part of a website is the body, this is where almost all of the code will go including images, text and input fields.

Basic Elements

The most common element used is the heading. It is represented by h1-5, h1 being the biggest going down through to h5. Another common element is the p element, which is the thing to use when writing out a paragraph.

There are other useful things to add which act as an alternative to using CSS, this being "b", which is used to bold something and "i" which is used to italic something.

A element that is less often used but is still a useful basic element is the "br" which is used to break the line between to elements. This is a self closing tag, meaning that it doesn't need another tag at the end with "/".

One more of the most common elements utilized is the image element, which is a self closing tag and inserts an image from your computer. The image must be within the same folder as the code.

Example

Another element similar to the image tag to master is that of links.To link text or an image to something else, start by putting in a "a" followed by href="" (all within the same tag) put the address of what you want to be linked to with in the quotation marks, then put your closing tag and whatever you wish to link betweent the two.

Advanced Elements

Lists are an essential skill to master in order to create a great looking website. The best part is is that they are reletively simple. To start a list all that is neccesary is create a tag with "ul" or "ol" and inside of this you can put as many options as you would like. "ul" creates an unordered list with just bullet points. While "ol" creates an ordered list involving a numbered list. After those tags are put in place each new item in the list much have the tags "li"

Input boxes serve to take in the users input and are great for making things such as serve forms. First things first is to put the "form" tag and then put a tag with "input" within your form. There are many different types of inputs such as:
